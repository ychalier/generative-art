instructions
First rule must be named E. There should be a rule named A containing terminal nodes only. Possible nodes are: x, y, bw (random 1d value), rgb (random 3d value), sin(A), cos(A), tan(A), exp(A), sqrt(A), sum(A, B), mult(A, B), mod(A, B), triple(A, B, C) (build 3d from 3 components), mix(A, B, C) (blend first two channels, third channel determines the balance), level(A, B, C) (yields second or third channel depending on a random threshold on first channel).
