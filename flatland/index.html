What is this:

A simulation of what you would see walking around Flatland.

How to use this:
  • Move the mouse to look around
  • Use the arrows keys (←↑↓→) or ZQSD (AZERTY layout) to move around
  • Hold the SHIFT key to move faster
  • Hold the CTRL key and move the mouse to move the light around
  • Use the mouse wheel to increase/decrease the FOV
  • Use the 'B' key to toggle bird's-eye view
  • Press the 'H' key to return to the starting position
  • Click anywhere to start animation
